### Basic HTML Tags

```html
<html> Root of HTML Document

<head> Info about Document

<body> Document Body
    
<h1>,<h2>,<h3> Header tags
    
<p> Paragraph tag

<div> Generic block section tag 容器

<span> Generic inline section tag
```
